\section*{A5.36}

\begin{enumerate}
\item If $\lambda^\star$ is large, then tightening the constraint
(decreasing $s$) causes the optimal value to increase a lot.

\item If $\lambda^\star$ is large, then loosening the constraint
(increasing $s$) causes the optimal value to decrease a lot.

\item If $\lambda^\star = 0$, then optimal value is unchanged.
%Small changes in $s$ do not affect the optimal value to first order, though
%the optimal value may decrease if the constraint is loosened sufficiently.
\end{enumerate}
