\section*{A6.2 Minimax rational fit to the exponential}

We consider the specific problem instance with data
\[
t_i = -3 + \frac{6(i-1)}{k-1}, \quad y_i = e^{t_i}, \quad i = 1, \ldots, k,
\]
where $k = 201$. (In other words, the data are obtained by uniformly sampling the exponential function over the interval $[-3, 3]$.)

Find a function of the form
\[
f(t) = \frac{a_0 + a_1 t + a_2 t^2}{1 + b_1 t + b_2 t^2}
\]
that minimizes $\max_{i=1,\ldots,k} |f(t_i) - y_i|$. (We require that $1 + b_1 t_i + b_2 t_i^2 > 0$ for $i = 1, \ldots, k$.)

\textbf{Solution.}

As shown in problem 6.9, this is a \textbf{quasiconvex optimization problem}. We solve it using \textbf{bisection} on the objective value $\gamma$, combined with a feasibility LP at each step.

For a given $\gamma > 0$, we check feasibility of:
\[
\left| \frac{p(t_i)}{q(t_i)} - y_i \right| \leq \gamma, \quad i = 1, \ldots, k
\]
where $p(t) = a_0 + a_1 t + a_2 t^2$ and $q(t) = 1 + b_1 t + b_2 t^2$.

Since $q(t_i) > 0$, multiplying by $q(t_i)$ gives the linear constraints:
\[
-\gamma \, q(t_i) \leq p(t_i) - y_i \, q(t_i) \leq \gamma \, q(t_i), \quad i = 1, \ldots, k
\]

Expanding:
\[
-\gamma (1 + b_1 t_i + b_2 t_i^2) \leq (a_0 + a_1 t_i + a_2 t_i^2) - y_i (1 + b_1 t_i + b_2 t_i^2) \leq \gamma (1 + b_1 t_i + b_2 t_i^2)
\]

Rearranging, for each $i$:
\[
a_0 + a_1 t_i + a_2 t_i^2 - (y_i + \gamma) b_1 t_i - (y_i + \gamma) b_2 t_i^2 \leq y_i + \gamma
\]
\[
a_0 + a_1 t_i + a_2 t_i^2 - (y_i - \gamma) b_1 t_i - (y_i - \gamma) b_2 t_i^2 \geq y_i - \gamma
\]

These are linear in the variables $(a_0, a_1, a_2, b_1, b_2)$.

We also need $q(t_i) > 0$, which we enforce as $q(t_i) \geq \epsilon$ for a small $\epsilon > 0$.

\textbf{Results.}

Using bisection with tolerance $0.001$, we obtain:
\begin{align*}
a_0 &= 1.0099 \\
a_1 &= 0.6117 \\
a_2 &= 0.1134 \\
b_1 &= -0.4147 \\
b_2 &= 0.0485
\end{align*}

\textbf{Optimal objective value:} $\gamma^\star \approx 0.0233$
